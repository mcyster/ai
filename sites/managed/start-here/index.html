Change a Website

Click on the "Open Chat" button to talk to the AI Web Developer.

Note: It can take 2 or more minutes for the page to update after submitting a prompt.

Prompts to change a website:

Please update the page to give a brief description of Beep Boop
Please update the background color of the page to light blue
Please add 100 bouncing balls to the page
I would like to make a simple space invaders game, please update the page to implement a basic space invaders game using javascript
Can you add an interactive form to the page to collect user feedback?
Please embed a YouTube video tutorial on how to use this AI assistant

If an javacsript error occours on your website an error button will appear on the chat window, click on that button to get the error in the chat prompt so you can send it to the AI to debug.

The chat window supports <shift><enter> to add a new line when you are typing a message.

Manage a Website

Its generally best to make a copy of a website before you change it, ask the AI to copy the page before you start working on it.

The website name is derived from the title of the page, ask the AI to change the title to update the name of the website on the listing page.

Ask the AI to add tags to the page, so you can find the page in the future

Special reserved tags (if you add these tags to your website, they will be ignored):

  • start - for this start website / page
  • curated - specified a well defined set of example pages
  • managed - used websites not being managed in revision control
  • unmanaged - websites not being managed in revision control

Prompts to manage a website:

Please make a copy of this website
Please add the tag: example
Please remove the tag: example
Please change the title of the page to: example
